# Chord Game Engine

A side project to learn OpenGL, rendering, game engine design and to use C++ in a larger codebase.

# Current Progress:

- [x] 3D object rendering
- rotations
- translations
- scaling
- texture mapping

- [x] Camera Control
- Mouse and keyboard (WASD)

- [x] Lighting
- Ambient lighting
- Diffuse lighting
- Specular lighting
- Point lights
- Spot lights
